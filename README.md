## Use

* Python 3.12
* Flask 3.0.3
* mysql-connector-python 8.4.0
* Faker (Generates fake data)


## installation
```bash
sudo apt-get install python3.12
sudo apt-get install python3-pip

```bash
pip install -r requirements.txt
python3 migrations.py
```
## Run

```bash
python3 app.py
```
